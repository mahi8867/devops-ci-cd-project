🚀 DevOps CI/CD Project

📌 Project Overview

This project demonstrates an end-to-end CI/CD pipeline using:
GitHub → Jenkins → Docker → Kubernetes

---

⚙️ Tools & Technologies

- GitHub (Source Code Management)
- Jenkins (CI/CD Automation)
- Docker (Containerization)
- Kubernetes (Container Orchestration)

---

🔄 CI/CD Pipeline Flow

1. Developer pushes code to GitHub
2. Jenkins triggers build automatically
3. Docker image is built
4. Image is pushed to Docker Hub
5. Application is deployed to Kubernetes

---

📁 Project Structure

- "Dockerfile" → Build Docker image
- "Jenkinsfile" → CI/CD pipeline script
- "deployment.yaml" → Kubernetes deployment
- "service.yaml" → Kubernetes service

---

Project setup in progress

🎯 Goal

To build a real-world DevOps pipeline and deploy applications automatically
